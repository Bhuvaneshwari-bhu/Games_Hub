
# 🎮 Games Hub

[Live Demo](https://bhu-games-hub.netlify.app)

A browser-based **multi-game web application** featuring Tic Tac Toe, Memory Card Game, Snake & Ladders, Sliding Puzzle, and Reaction Time Game.  
Track scores and stats using **localStorage** for persistent gameplay.

---

## 🕹️ Features

- **Tic Tac Toe** – Track wins, losses, and draws.  
- **Memory Card Game** – Best moves and total games played.  
- **Snake & Ladders** – Player win counts tracked.  
- **Sliding Puzzle** – Count of solved puzzles.  
- **Reaction Time Game** – Track fastest reaction time.  
- **Persistent Stats** – Data saved locally using localStorage.  
- **Responsive Design** – Works on mobile and desktop.  
- **Clean UI** – Simple and intuitive game selection interface.

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Netlify for deployment
- Claude code

---

## 🚀 How to Run Locally

1. Clone the repo:

```bash
git clone https://github.com/Bhuvaneshwari-bhu/Games_Hub.git
````

2. Open `index.html` in a browser.
3. Start playing! Stats are saved automatically.
